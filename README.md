# SCT_DS_2
# Task 2 - Data Cleaning

## Overview
This project is part of the Data Science Internship at SkillCraft Technology.

The objective of Task 2 is to clean and preprocess raw datasets by handling missing values, duplicates, and inconsistent data for better analysis.

---

## Objectives
- Identify missing values
- Remove duplicate records
- Clean inconsistent data
- Prepare datasets for analysis

---

## Tools & Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib

---

## Dataset Used
The following datasets were used:

- Tips Dataset
- Flights Dataset

Datasets are stored inside the `data/` folder.
